I am ___.

This piece features multiple self-referential objects placed in a room together to create a novel virtual space. There is an accompanying augmented reality application with the piece which allows users to insert virtual objects into this space depending on their location with respect to certain visual triggers. The piece contains a diverse set of references to major themes in the transition from modern art to the contemporary state of art. It hopes to create an emergent piece of art whose only context is art in attempt to create a tautologically-sound artwork.
